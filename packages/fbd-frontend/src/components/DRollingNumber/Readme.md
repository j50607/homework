# DRollingNumber 使用方法

## 如何使用?

在該頁將元件引入

import DRollingNumber from '@/components/DRollingNumber'

components: {
  DRollingNumber
}

## 資料

# num 類型為 Number

num傳入會一直改變的數字。

# width 類型為 Number

預設為24，會影響字體大小。

# height 類型為 Number

預設為24，會影響數字line-height。

# color 類型為 String

預設為#000，會影響數字顏色。

# transition 類型為 Number

預設為0.6，會影響轉動速度。
